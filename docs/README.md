# Boilerplate Docs

This directory contains recurring workflow guides for the repository. The
implemented contract lives in [`../spec/README.md`](../spec/README.md). The
authoritative terminology lives in [`../glossary/README.md`](../glossary/README.md).
Future-facing follow-up lives in [`../notes.md`](../notes.md).

## Guides

- [`./context-system/README.md`](./context-system/README.md): task briefing, locked-subsystem policy, generated artifacts, and review packets
- [`./repo-workflow/README.md`](./repo-workflow/README.md): repo-wide git and `notes.md` workflow rules

## Rule Of Thumb

- Reach for `spec/` when you need current behavior or workflow rules.
- Reach for `glossary/` when exact terminology matters.
- Reach for `docs/context-system/` when you need the anti-drift workflow.
- Reach for `docs/repo-workflow/` when you need repo-wide contributor rules.
